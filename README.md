# 🃏 Technorift Card System

> A dynamic, fully responsive card management system built entirely from scratch using Core PHP and MySQL.

## 🚀 Project Overview
Developed as a task for the Technorift internship program, this project demonstrates a strong foundation in raw web development. Instead of relying on frameworks, this application handles database connections, data fetching, and UI rendering through completely handwritten PHP and SQL.

## 💻 Tech Stack
* **Frontend:** HTML5, CSS3 (Custom handwritten styling)
* **Backend:** Core PHP
* **Database:** MySQL

## ✨ Key Features
* **Dynamic Data Rendering:** Card content is pulled directly from a MySQL database using custom SQL queries.
* **Zero Frameworks:** Demonstrates a raw understanding of backend mechanics and PHP fundamentals.
* **Fully Responsive:** The CSS layout adapts seamlessly to desktop, tablet, and mobile screens.

## 🛠️ Local Installation
1. Clone the repository:
   `git clone https://github.com/bilalifzal/TechnoRift-Card-System.git`
2. Move the project folder to your local server directory (e.g., `htdocs` for XAMPP).
3. Open phpMyAdmin and create a new database.
4. Import the provided `database.sql` file to set up the required tables.
5. Update the connection credentials in your `db_connect.php` file.
6. Open the project in your browser: `http://localhost/TechnoRift-Card-System`
